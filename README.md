# Analysis Scripts: (f) Divergence and Selection in Rearrangements

This repository contains the scripts used to perform the analysis for the section **"(f) Divergence and selection in rearrangements"** of the article.

## 📋 Prerequisites & Inputs
To run these scripts, you will need:
-Protein files (`.aa`) and GFF3 file (.gff3) generated by **Braker** for both genomes.
-Clean genome assemblies of both genomes (.fasta)
-Synteny hits(.synHits.txt) output from GENESPSACE 

## Analysis Workflow
The scripts provided here demonstrate the analysis pipeline for one specific rearrangement: 
* **Target:** `trans_chr24`

> **Note:** Although the examples shown focus on `trans_chr24`, this same pipeline was applied to all rearrangements discussed in the paper.

## Usage
These scripts are designed to be executed on a **High-Performance Computing (HPC)** cluster using the **Slurm** workload manager. 

The workflow is divided into three main stages:

1. **Internal & External Analysis:** Run the two master Slurm scripts. It is highly recommended to run these in **separate folders** to avoid file overwriting:
   - `master_inside.sh`: Calls 9 Python scripts to analyze genes *inside* the breakpoint/inverted area.
   - `master_outside.sh`: Calls 9 Python scripts to analyze genes *outside* the area.

2. **Comparison:** - Once both analyses are complete, run the final Python script (e.g., `compare_results.py`) to merge and compare the data from both areas.

3. **Results:**
   - The output from the comparison script was used to construct **Table 2** and **Table S2** of the article
